About

Temtem NPC map is a fan-made map for the game Temtem. In Temtem patch 0.6.12, the game introduced a Postal Service mechanic in which the player is given 1-3 parcels and tasked with delivering them to NPCs all over the game world! The goal of this interactive map is to make it a little easier to find these NPCs.
